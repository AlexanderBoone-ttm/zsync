Acknowledgements

zsync uses the rsync algorithm, from the technical paper by Andrew Tridgell and Paul Mackerras.

zsync also contains some code from zlib, Copyright (C) 1995-2003 Jean-loup Gailly and Mark Adler. I have made only slight alterations to this code, which I have put in the public domain.

zsync also includes checksum code taken from OpenBSD. The MD4 code is public domanain, by Colin Plumb and Todd C. Miller. The SHA1 code is also public domain, by Steve Reid. This code is in the libhash/ subdirectory, see the individual files for their non-copyright notices.

I would like to thank the Free Software Foundation and its contributors, for gcc, gdb and emacs, the essential development tools. Also I would thank Sourceforge for providing the hosting facilities for the downloads and mailing list, and the compile farm for testing portability.
